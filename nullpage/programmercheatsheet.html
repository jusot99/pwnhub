Programmer Reference Cheat Sheet

A handy reference for coding, debugging, and mastering essential programming concepts.

1. Command Line Essentials

# Navigate directories
cd [directory]

# List files
ls -la

# Create a file
touch [filename]

# View file contents
cat [filename]

2. Git Commands

# Clone a repository
git clone [url]

# Check status
git status

# Stage changes
git add .

# Commit changes
git commit -m "Commit message"

# Push to repository
git push origin [branch]

3. Python Snippets

# Basic loop
for i in range(5):
    print(i)

# List comprehension
squares = [x**2 for x in range(10)]

# Define a function
def greet(name):
    return f"Hello, {name}!"

4. JavaScript Snippets

// Basic loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// Array map
const squares = [1, 2, 3].map(x => x * x);

// Arrow function
const greet = name => `Hello, ${name}!`;

5. SQL Queries

-- Select all records
SELECT * FROM table_name;

-- Insert data
INSERT INTO table_name (column1, column2) VALUES (value1, value2);

-- Update data
UPDATE table_name SET column1 = value1 WHERE condition;

-- Delete data
DELETE FROM table_name WHERE condition;

6. Debugging Tips

"Debugging is twice as hard as writing the code in the first place." - Brian Kernighan
  • Use print statements or logging to trace code execution.
  • Break down the problem into smaller parts.
  • Leverage tools like debuggers and profilers.
