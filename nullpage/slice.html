mitmproxy Playground

Use this page to practice intercepting HTTPS, editing requests, and inspecting WebSockets.
Tip: Serve this file via a local server so your browser uses the system proxy. For example: python3 -m http.server 8000 then open http://127.0.0.1:8000/index.html.

How to use with mitmproxy / mitmweb

      
    
        
  1. Install: pip install mitmproxy (or your package manager).
    2. 
        
  2. Start UI: mitmweb -p 8080 (web UI) or mitmproxy -p 8080 (TUI).
    3. 
        
  3. Set your system/browser proxy to HTTP/HTTPS: 127.0.0.1:8080.
    4. 
        
  4. Open http://mitm.it in the proxied browser → download & install the mitmproxy CA certificate for your OS/browser.
    5. 
        
  5. Reload this page and click the buttons above. You should now see decrypted HTTPS flows.
    6. 
        
  6. To intercept/pause flows, press i in mitmproxy or enable intercept in mitmweb → edit → resume.
    7. 
        
  7. To rewrite on the fly: mitmweb → Options → Addons → Modify body/headers or run mitmproxy with a small addon script.
    8. 
      

      

      

        Example addon (save as edit_title.py)

from mitmproxy import http

def response(flow: http.HTTPFlow) -> None:
    if flow.response and flow.response.headers.get("content-type", "").startswith("application/json"):
        flow.response.headers["x-mitmproxy-playground"] = "✅ touched"


        
Run with: mitmproxy -s edit_title.py -p 8080

      

    

  


  

    
Security note: Only intercept traffic you own or have explicit permission to test. Installing a custom CA gives the proxy power to decrypt your HTTPS. Remove it when done.
